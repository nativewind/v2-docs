# Migrations to v4

## New setup

Please follow the instructions in the relevant Getting Started guide. The setup has slightly changed.

## `styled()` has been removed


Previous to v4, NativeWind used to change `className`->`style` 'inplace'. Meaning it transformed that component directly. In v4, `className` is now passed down as a prop until it reaches a core React Native component (e.g `View`/`Text`)

`styled()` has been removed as it is larger
